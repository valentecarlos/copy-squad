### Halbert: contexto que dev precisa antes de pesquisar

> **Source:** conhecimento público sobre copywriting + sources to be fetched

**Quem foi Gary C. Halbert (1938-2007):**
- "The Prince of Print" — copywriter americano de direct response
- Famoso por sales letters de centenas de milhões de dólares vendidos
- Fundador da newsletter "The Gary Halbert Letter" (1986+)
- Encarcerado em 1984 por fraude postal — escreveu "The Boron Letters" (cartas semanais ao filho Bond, depois publicadas como livro de copywriting)
- Mentorou geração de copywriters direct response modernos (Carlton, Bencivenga, Kennedy)

**Peças canônicas conhecidas (a serem detalhadas em exemplos.md):**
- **The Coat-of-Arms Letter** (anos 70) — sales letter para venda de "brasões de família"; gerou US$ 7 milhões em vendas (era moeda forte na época)
- **Tova Borgnine** — sales letter para perfume da então-esposa de Ernest Borgnine; vendas explosivas
- **The Killer Ad** — anúncio de jornal para escola de inglês ou produto similar
- Conteúdo do Newsletter (boletim de assinatura)

**Frameworks que Halbert popularizou:**
- **A-pile / B-pile** — toda correspondência cai em uma das duas pilhas (interessante = A, lixo = B); copy precisa engenhosar pra entrar na A
- **Starving crowd** — escolha do mercado faminto antes do produto: "se eu te der R$ 1 milhão pra abrir um restaurante e R$ 1 milhão pra encontrar uma multidão faminta, qual você prioriza?"
- **Motion beats meditation** — agir/testar > planejar/teorizar; ethos de direct response
- **The Boron Letters** — princípios de copywriting + vida apresentados como cartas de pai pra filho

### `validate-research-format.js` schema

> **Source:** Architecture §5.3 Job 3 + DEC-002 (Architecture §3.4)

```javascript
#!/usr/bin/env node
'use strict';

const fs = require('fs');
const path = require('path');
const { listMarkdownFiles } = require('./lib/markdown-utils');

const SOURCES_REGEX = /^\d+\.\s\*\*[^*]+\*\*\s—\s(https?:\/\/\S+)\s\(accessed\s(\d{4}-\d{2}-\d{2})\)\s—\s.+$/;
const MIN_SOURCES = 3;
const MIN_DATE = '2024-01-01';

function validateFile(filePath, content) {
  const violations = [];
  const fontesMatch = content.match(/##\s+Fontes\s*\n([\s\S]+?)(?=\n##\s|$)/i);
  if (!fontesMatch) {
    violations.push('Section "## Fontes" não encontrada');
    return violations;
  }

  const fontesContent = fontesMatch[1];
  const items = fontesContent.split('\n').filter(l => /^\d+\./.test(l.trim()));

  if (items.length < MIN_SOURCES) {
    violations.push(`Apenas ${items.length} fontes (esperado ≥${MIN_SOURCES})`);
  }

  for (const item of items) {
    const m = item.trim().match(SOURCES_REGEX);
    if (!m) {
      violations.push(`Fonte mal-formatada: ${item.substring(0, 80)}...`);
      continue;
    }
    const [, url, dateStr] = m;
    const date = new Date(dateStr);
    const today = new Date();
    if (isNaN(date) || date < new Date(MIN_DATE) || date > today) {
      violations.push(`Data inválida em fonte: ${dateStr}`);
    }
  }

  return violations;
}

// ... main() segue padrão de validate-structure.js
```
