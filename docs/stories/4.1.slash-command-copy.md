## Dev Notes

### Anthropic slash command format (per architecture.md §1.1 + Anthropic docs)

```markdown
---
description: "Short description of what this command does"
argument-hint: "<expected arguments format>"
---

[Body — markdown content invoking agent or providing context]
```

Body pode incluir:
- `@agent-name $ARGUMENTS` — invoca agent passando user args
- Exemplos de uso
- Notas para Claude Code processing

### Description distintivo

`/copy` é o entry point oficial do **Copy Squad**. Claude Code apresenta description em command palette (Cmd+K), então deve ser:
- **Curta** (1 linha)
- **Clara** sobre when to use
- **Distintiva** vs. outros comandos do harness Claude Code

Exemplo: "Aciona o Copy Squad — 15 copywriters lendários como subagentes orquestrados por Copy Master para produzir copy direct response ou brand."

### Argument-hint

`argument-hint` é texto sugestivo mostrado ao usuário ao começar a digitar `/copy`. Deve indicar formato esperado:
- `<descrição da demanda em 1-2 linhas>` é mínimo
- Pode incluir exemplos: `"VSL para curso de yoga 35+ R$ 297"`

### Exemplos a documentar (≥3)

1. **VSL / Sales Letter long-form**:
   ```
   /copy "VSL de 8min para curso de jiu-jitsu para iniciantes 35+, R$ 497, mercado problem-aware"
   ```
   Esperado: Copy Master roteia para 2-4 copywriters DR (provavelmente Halbert + Sugarman + Bencivenga), produz draft + critique + 2 versões finais

2. **Headlines / Capture page**:
   ```
   /copy "20 headlines variants para landing page de e-book gratuito sobre produtividade matinal"
   ```
   Esperado: Copy Master invoca Caples (headline-craft) + Sugarman (slippery slide) + provê 20 variants categorized

3. **Brand storytelling / positioning**:
   ```
   /copy "Brand storytelling para nova linha premium de cafés especiais — refletindo origem fazenda + ritual matinal"
   ```
   Esperado: Copy Master invoca brand vertical (Ogilvy + Burnett + Abbott) + entrega manifesto de marca + 3 variations de tone

### Dependency note (FR15)

Slash command 4.1 invoca `@copy-master` que será criado em **Story 4.5**. Por ora, o command file é estruturalmente válido + frontmatter parseable + body referencing `@copy-master` funcionará automaticamente quando 4.5 entregar o agent. Não há circular dependency — Anthropic resolve `@agent-name` em runtime quando comando é invocado.

### Word count target

Slash command files **não têm NFR1 ≥600 words** (NFR1 aplica a agent system prompts). Command body pode ser curto (50-300 words) — focado em invocation + examples.

### Technical Constraints

- Frontmatter YAML parseable
- Body contém `@copy-master $ARGUMENTS` invocation
- Encoding UTF-8 + LF + sem BOM
- PT-BR para examples + descriptions (NFR9)
