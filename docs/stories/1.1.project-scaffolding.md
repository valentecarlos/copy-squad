# Story 1.1: Project scaffolding

## Status

**Draft**

## Story

**As a** developer,
**I want** estrutura de pastas e arquivos base criada no repositório,
**so that** os epics seguintes têm onde colocar agentes, pesquisa, scripts e testes sem decidir layout no meio do caminho.

1. Pasta `research/` criada na raiz com `.gitkeep`
2. Pasta `.claude/agents/_shared/` criada com `.gitkeep`
3. Pasta `scripts/` criada com `.gitkeep`
4. Pasta `tests/install/` criada com `.gitkeep`
5. Pasta `docs/architecture/` criada com `.gitkeep`

### `package.json` schema mínimo

> **Source:** `docs/architecture.md#85-javascript-nodejs-standards` + ADR-021

```json
{
  "name": "copy-squad",
  "version": "2.0.0-draft.1",
  "private": true,
  "description": "Copy Squad — 15 copywriters lendários como subagentes Claude Code, orquestrados via Copy Master",
  "repository": {
    "type": "git",
    "url": "https://github.com/valentecarlos/copy-squad"
  },
  "license": "MIT",
  "engines": {
    "node": ">=20"
  }
}
```

**Justificativas:**
- `name: "copy-squad"` — kebab-case (Architecture §8.6), corresponde ao slug GitHub
- `version: "2.0.0-draft.1"` — semver pré-release (Architecture §6.6)
- `private: true` — não publica em npm yet (npm package fica para v2.3, ADR-007)
- Sem `dependencies` ou `devDependencies` — Story 1.2 adicionará `js-yaml@^4.1.0` (ADR-021); zero deps obrigatórias hoje
- `engines.node: ">=20"` — Node 20 LTS (ADR-021)

### Naming conventions a respeitar

> **Source:** `docs/architecture.md#86-naming-conventions`

- Arquivos `.gitkeep`: literal `.gitkeep` (ponto-prefixo, lowercase, sem extensão)
- `package.json`: literal lowercase
- `folder-structure.md`: kebab-case
- Bash/JS files (futuras stories): seguirão respectivas convenções

### Encoding (NÃO violar — Critical Rule #5)

> **Source:** `docs/architecture.md#38-encoding--line-endings`

- **UTF-8** em todos os arquivos criados
- **LF** line endings (`\n`) — **proibido** CRLF
- **No BOM** (`\xEF\xBB\xBF`) — **proibido**
- **Diacríticos PT-BR** preservados literalmente em `folder-structure.md` (NUNCA `nao` por `não`)

> Validation manual nesta story (CI ainda não rodando até Story 1.4):
> ```bash
> file -bi docs/architecture/folder-structure.md  # deve retornar charset=utf-8
> file -bi package.json  # idem
> ```
