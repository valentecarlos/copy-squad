### Fontes promissoras descartadas

- **Amazon book listings** (Mott biography) — descartadas por serem páginas comerciais sem conteúdo extraível para citação
- **Goodreads page** — overview superficial, sem detalhes para citar
- **Grokipedia** — fonte alternativa não verificável, preferível usar Wikipedia direta ou sites independentes
- **PDF Coffee Coat-of-Arms PDF** — provavelmente cópia não autorizada; preferível usar swiped.co (curatorial legítimo)
- **YouTube playlists** — usados apenas como referência; não acessei conteúdo de vídeo (limitação de WebFetch para vídeo)
